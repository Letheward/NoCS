> Common ET

`F` = Target Frequency  
`f` = Base Frequency  
`a` = Scale Number (integer)  
`x` = Degree       (integer)  

In a octave:

`F = f * ( 2 ^ ( x / a ))`

## Linear Equal Temperament (LET) 

> New ET 

`F` = Target Frequency  
`f` = Base Frequency  
`a` = Scale Number (integer)  
`x` = Degree       (integer)  

In a octave:

`F = f * ( 1 + x / a )`
